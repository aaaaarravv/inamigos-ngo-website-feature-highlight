# NGO Website Feature Highlight – InAmigos Foundation

## Overview

This project was created as part of a task to identify and visually present essential features that can improve an NGO website's usability, engagement, and impact.

## Features Included

### 1. Donation Section

Provides a simple and accessible way for supporters to contribute financially to the organization's mission.

### 2. Volunteer Registration

Allows interested individuals to register and participate in NGO activities and events.

### 3. Impact Showcase

Highlights key achievements and statistics to demonstrate transparency and organizational impact.

### 4. Ongoing Campaigns

Promotes current initiatives and encourages community participation.

### 5. Testimonials

Features feedback from volunteers and beneficiaries to build trust and credibility.

## Tools Used

* Figma
* GitHub

## Objective

To understand the essential components of an NGO website and propose features that can enhance user engagement, transparency, and social impact.
